# Happy Birthday Aafreen - Website

This is a dedicated birthday website for Aafreen Khan, created by Prabhav Fulara.

## How to Setup

1.  **Add Audio Files**:
    *   Go to the `assets` folder.
    *   Add your Happy Birthday song and name it `hbd_song.mp3`.
    *   Add your voice message and name it `prabhav_message.mp3`.

2.  **Add Photos**:
    *   Add photos to the `assets` folder.
    *   Open `index.html` and replace the placeholder `div`s in the `.gallery-grid` section with actual `<img>` tags.
    *   Example: `<div class="gallery-item"><img src="assets/photo1.jpg" alt="Memory"></div>`

3.  **Deploy to GitHub Pages**:
    *   Create a new repository on GitHub.
    *   Upload all these files to the repository.
    *   Go to Repository Settings -> Pages.
    *   Select `main` branch and save.
    *   Your website will be live!

## Features
*   **3D Card Intro**: Tap to open the envelope.
*   **Music**: Plays automatically after opening the card.
*   **Voice Note**: Dedicated button to play your personal message.
*   **Timeline**: A journey of your relationship.
